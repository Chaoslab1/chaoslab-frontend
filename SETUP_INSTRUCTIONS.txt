🎯 WHAT IS THIS?
  This is the DOCUMENTATION PORTAL (frontend) for the Chaoslab API.
  It has been separated from the backend API codebase.

📦 WHAT'S INCLUDED?
  ✓ Docusaurus documentation site
  ✓ 68 markdown documentation files
  ✓ React components (API Reference, GraphQL Playground)
  ✓ Multi-language support (English, French)
  ✓ All static assets and configurations

🚀 QUICK START

  Step 1: Install Dependencies (First Time Only)
  ────────────────────────────────────────────────
  cd C:\Users\user\Documents\Chaoslab\chaoslab-frontend-temp
  npm install

  (This will take 2-3 minutes)


  Step 2: Start Development Server
  ────────────────────────────────────────────────
  npm start

  The site will open automatically at: http://localhost:3001


  Step 3: Start Backend API (In Separate Terminal)
  ────────────────────────────────────────────────
  cd C:\Users\user\Documents\Chaoslab\chaoslab
  npm run dev

  The API will run at: http://localhost:3000

🆘 TROUBLESHOOTING

  Problem: npm install fails
  Solution:
    npm cache clean --force
    del package-lock.json
    npm install

  Problem: Port 3001 in use
  Solution:
    Change port in package.json:
    "start": "docusaurus start --port 3002"

  Problem: Changes not appearing
  Solution:
    - Clear cache: del /s /q .docusaurus
    - Restart: Ctrl+C then npm start

  Problem: Build fails
  Solution:
    del /s /q node_modules .docusaurus
    npm install
    npm run build
